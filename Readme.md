this project is a simple programmatic interaction with aave which involves
- supplying collateral
- borrowing assets
- repaying them back witb either a tokens or the borrowed asset itself

LendV3 is python scripts interacting with aave v3 while lend for aave v2, their repective 
interfaces are found on the interface section.

LendV3 script is mainly tested on goerli testnet, it was also supposed to be tested on 
polygon-main-fork, there was issues encountered, still looking into it.

lend v2 was testes using the deprecated kovan testnet and mainnet fork.
